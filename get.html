Get FluentWeb

Let's get you some Fluent design without touching React! Here is how you get started:

1. Base CSS

The first thing you're gonna need is the CSS itself. Download it here. If it opens a blank page, copy the code, create a new file called styles.css and paste it in.

2. New UI elements

If you want some new UI elements, download elements.js by clicking this link. Same as the CSS, if it doesn't download, copy the code and paste it into a new file called elements.js.

3. Including them

Just adding FluentWeb isn't enough - you need to include it. Like this:

<link rel="stylesheet" href="styles.css">
<script src="elements.js"></script>   <!-- Ignore this one if you don't download elements.js -->
