Start using FluentWeb

To make your site look a bit fluent-ier, please check out the download page.

If you want a quick start template for your page, here is one:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <link rel="stylesheet" href="styles.css">
  <link rel="icon" href="logo.png" type="images/png">
  <script src="elements.js"></script>
  <title>|</title>
</head>
