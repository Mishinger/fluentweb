Layout

Main, body, header and sidebar

Here is how these work in FluentWeb:
  • Use class="image" on body to make it the --image set in the CSS file.
  • Main with a class="main-card" to make it a card - great over images like the <body class="image"> thingy.
  • If you want a sidebar, use a div with the sidebar class. Note that it must be in the body before the main content and the main must have an after-sidebar class. If you prefer a compact sidebar, consider using the class compact class on the sidebar div alongside the sidebar class and replacing the main's after-sidebar class with after-compact-sidebar.
