Customize the design

Of course, the blue may not fit every app. Nor the styles may not be enough. So that's why there is this page. Here is what you can tweak:

1. Accent color

Change the 3 --accent-color variables to your brand color. The one in the defauult root and print should be the same, while the dark mode one can (but doesn't need to be) a bit lighter.

2. New styles

Want a shiny golden button for subscribing to the pro version of your app? Sure, do that - like this:

  1. Create a new color in the three roots - e.g. --pro-color: #fd1; for the light and print with --pro-color: #fd5; being the dark mode version.
  2. Add the style - with this button example, it's like this: 
    button.pro {
  color: var(--accent-text);
  background-color: var(--success-color);
  border-color: var(--success-color);
}
button.pro:hover {
  background-color: color-mix(in srgb, var(--pro-color) 85%, var(--text-color) 15%);
  border-color: color-mix(in srgb, var(--pro-color) 85%, var(--text-color) 15%);
}
3. Changing the styles

Not reccomended

You may break the Fluent design while tweaking the existing elements

If you are a power user, you can change the styles to your liking.

4. Inline styles

You can use inline styles to change things like layouts or add specific colors using <p style="color: #07f">Yay, I'm cyan!</>

5. Image

Set the --image variable to an image or gradient that will be displayed on background images (more on that later in Elements > Layout)
